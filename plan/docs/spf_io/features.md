# spf.io Product Features

Spf.io provides a suite of tools categorized into three main pillars: **Events**, **Content**, and **Conversations**.

## 1. Events: Live Captioning & Translation
The flagship offering for real-time engagement at events.

### Automatic Captions & Translation
- **Purpose**: Real-time accessibility and inclusion for live speech.
- **Key Benefits**:
    - **Accessibility**: Real-time captioning for deaf and hard-of-hearing audiences.
    - **Inclusion**: Instant translation in 60+ languages.
    - **Scalability**: Supports thousands of concurrent users on individual mobile devices.
- **How it Works**: Uses AI to generate captions from a live audio feed. A "human-in-the-loop" (Director) can monitor and edit the AI output in real-time to ensure 100% accuracy for names or technical terms.
- **Supported Platforms**: Zoom, YouTube Live, Microsoft Teams, Webex, and generic RTMP streaming.

### Audio Live Streaming
- **Purpose**: Low-latency audio translation for listeners.
- **Key Benefits**:
    - **Multilingual Listening**: Audiences can listen to synthesized AI voices or human interpreters.
    - **Device Agnostic**: Users listen via their own smartphones through a web browser; no app download required.
- **How it Works**: Captures source audio, translates it, and generates a high-quality synthesized speech stream.

### Multilingual Polls
- **Purpose**: Interactive audience engagement across language barriers.
- **Key Features**: Create polls that are automatically translated into the audience's preferred language, allowing everyone to participate seamlessly.

## 2. Content: Post-Production & Documents
Tools for archival and written material translation.

### Video Subtitling & Transcription
- **Purpose**: Making VOD and recorded content accessible and searchable.
- **Key Benefits**:
    - **SEO & Searchability**: Transcripts make video content searchable.
    - **Engagement**: Captions improve retention and allow viewing without sound.
- **How it Works**: Users upload video/audio files; the system generates transcripts and subtitle files (SRT, VTT) which can be edited and exported.

### Document Translation Portal
- **Purpose**: Batch translation of corporate and educational materials.
- **Key Benefits**:
    - **Format Preservation**: Maintains the original layout, images, and formatting of Word, PDF, and PowerPoint files.
    - **Custom AI**: The system learns from specific vocabulary and "Custom AI" models.
- **How it Works**: A drag-and-drop interface for batch translation with support for team collaboration and review.

### Slide Translation
- **Purpose**: Real-time or batch translation of presentation slides to ensure visual accessibility for multilingual audiences.

---

## 3. Conversations: Real-time Dialogue
Facilitating direct communication between individuals.

### Multilingual Conversations (Spf.io Convo)
- **Purpose**: 1-on-1 or small group dialogue where participants speak different languages.
- **Key Benefits**:
    - **Relationship Building**: Real-time bidirectional translation for speech and text.
    - **Versatility**: Works for in-person meetings, remote calls, or hybrid settings.
- **Supported Languages**: Over 40+ languages supported for bidirectional conversation.
- **How it Works**: A web-based chat and voice tool that translates input instantly between connected participants.
