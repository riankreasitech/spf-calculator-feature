# spf.io Research Summary

## Platform Overview
Spf.io is an all-in-one AI-powered translation and accessibility platform that provides real-time captioning and translation services in over 60 languages.

## Core Features
1. **Events (Live Streaming & In-Person)**
   - **Live Captions & Translation:** Real-time automatic captions visible on audience devices or big screens.
   - **Multilingual Polls:** Engage audiences in their native languages.
   - **Hybrid Workflow:** Human editors ("directors") can correct AI output in real-time.
2. **Content (Post-Production & Documents)**
   - **Video Subtitling:** Generate SRT/VTT files for VOD content.
   - **Document Translation Portal:** Translate slides, handouts, and scripts while preserving formatting.
3. **Conversations (1-on-1)**
   - **Multilingual Conversations:** Seamless dialogue between speakers of different languages for meetings.

## Pricing and Credits
- **Model:** Monthly or annual subscription based.
- **Entry Plans:** Generally range from **$99 to $199 USD per month** for small-to-midsize use cases.
- **Usage Credits:** Billed based on "minutes" consumed. Subscriptions include a base number of minutes; beyond that, overage charges apply.
- **Custom Setup:** One-time event quotes are available for users who don't need a recurring subscription.

## Technical Integrations
- **Zoom:** Uses Zoom's Caption API and a virtual audio cable (loopback) setup for bidirectional translation.
- **YouTube Live:** Supports real-time captions via YouTube's "caption ingestion URL" method.
- **Meeting Tools:** Native support or workarounds for Microsoft Teams, Google Meet, and Webex.
- **Event Platforms:** Integrates with Hopin, Crowdcast, and diverse web-based streaming tools.
